How to use the RPN calculator

This is an RPN calculator; the operator buttons
execute operations. E.g., to multiply 2 and 3, press
'2 ENTER 3 x' and the result 6.00 will appear.
All results show two decimal places - cents - as this is
a financial calculator.
Pressing the 'P' button and then a function button such as
'PV' will display a list of values to enter in the order to
be entered. To use the function, click the numbers to enter
the first value, click 'Enter'; continue for each required
value except the last - click the function button required
once the last value has been entered into the window,
and the results will appear.
Tool tip text will display when the mouse pointer is held
over a non-numeric button.
You can enter as many values as you like by clicking 'ENTER'
after each one; this will push all these values on to the stack.
Each operator button click thereafter will operate on the
value in the window and the next value on the stack, i.e.
'2 ENTER 3 ENTER + x' will produce 16.00
The 'STO' buttons will store the value in the window in the
register number on the button; to recall the value click 'RCL'
followed by the numeric button that corresponds to the
register number.
